## 📝 읽은 내용

### 전역 공간
- browser: 비엄격 모드에서 window
- Node.js: global
- ES Modules: undefined
### 함수 호출
- 비엄격 모드: window/global
- 엄격 모드: undefined
### 메서드 호출
- 마지막 점(.) 연산자 앞의 객체
### 생성자 함수 호출
- 새로 생성되는 인스턴스
### 화살표 함수
- 상위 스코프의 this
- call/apply/bind로 변경 불가

## 💡 인사이트
- 어딘가 애매한 this였지만, 규칙을 파악하니 헷갈리지 않을 수 있었다.
